## Project Overview

This editorial system was developed for a quarterly architecture journal focusing on sustainable urban design. The publication required a layout framework that could handle dense technical content alongside generous photographic spreads.

## Design Approach

The grid system is built on a 12-column baseline with a 4pt modular scale. This allows for 2, 3, 4, and 6 column variations while maintaining consistent vertical rhythm. The typographic hierarchy uses Playfair Display for headlines and Instrument Sans for body text, creating a dialogue between editorial tradition and contemporary clarity.

## Technical Details

- **Format:** 210 × 280 mm (A4)
- **Pages:** 128
- **Paper:** 120gsm matte coated
- **Binding:** Perfect bound
- **Colors:** 2 spot colors + black
