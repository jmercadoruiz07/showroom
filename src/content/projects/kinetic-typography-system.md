## Project Overview

A brand identity system where the logo and typography are inherently kinetic. The static state is merely a paused frame of a living system.

## Technical Approach

The system is built on three expression-driven rigs:

1. **Type Writer Rig:** Character-by-character reveal with configurable easing, stagger, and direction
2. **Form Follows Function:** Text boxes that resize and reflow based on content, with animated transitions between states
3. **Responsive Layout:** Compositions that adapt to aspect ratio changes via expression-driven layout engine

## Deliverables

- **Master Composition:** 1920×1080, 60fps, 30s duration
- **Social Cuts:** 9:16 (Reels/TikTok), 1:1 (Feed), 4:5 (Portrait)
- **Expression Library:** Reusable .jsx library for future projects
- **Documentation:** Notion workspace with usage guidelines
